# BNAppBar

The `BNAppBar` component provides a responsive application header with navigation tabs, user menu, and logo support. It seamlessly integrates with React Router for navigation.

## Features

- **Responsive Design**: Automatically adapts between desktop and mobile layouts. On mobile, tabs are converted to menu items in a drawer for better UX
- **Logo Support**: Flexible logo rendering with slots for components or images
- **Navigation Tabs**: Full React Router integration with custom Link components that work on both desktop and mobile
- **User Menu**: Avatar-based dropdown menu with navigation and action support
- **App Switcher Integration**: Support for application switcher components

## React Router Integration

### Tabs Navigation

- Use the `LinkComponent` prop to specify your router's Link component
- Add `to` props to tab objects for navigation paths
- **Important**: On mobile devices, tabs are converted to menu items in the drawer, so the `LinkComponent` is essential for proper navigation on all screen sizes

```tsx
import { Link } from 'react-router-dom'

;<BNAppBar
  LinkComponent={Link}
  tabs={[
    {
      label: 'Dashboard',
      value: 'dashboard',
      to: '/dashboard',
    },
    {
      label: 'Analytics',
      value: 'analytics',
      to: '/analytics',
    },
  ]}
/>
```

### Menu Items Navigation

- Use the `LinkComponent` prop to specify your router's Link component
- Add `to` props to menu item objects for navigation paths
- Menu items without `to` props will act as action buttons using `onClick`

```tsx
import { Link } from 'react-router'

const menuItems = [
  {
    label: 'Profile',
    to: '/profile',
    icon: <PersonIcon />,
  },
  {
    label: 'Settings',
    to: '/settings',
    icon: <SettingsIcon />,
  },
  {
    label: 'Logout',
    onClick: () => handleLogout(),
    icon: <LogoutIcon />,
  },
]

<BNAppBar
  LinkComponent={Link}
  menuItems={menuItems}
/>
```
