# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-19

### Added

- MUI-style color API for BNLogo component with support for 'primary', 'secondary', 'inherit', 'white', and custom color values (logo-color-api, 2024-12-19)
- Backward compatibility support for deprecated `logoFill` prop in BNLogo with development-mode deprecation warnings (logo-backward-compat, 2024-12-19)

### Changed

- BNAppFooter component updated to use new `color="white"` prop instead of deprecated variant approach (footer-logo-color, 2024-12-19)

### Documentation

- Updated BNLogo component interface with proper TypeScript deprecation annotations for `logoFill` prop (logo-docs, 2024-12-19)
