# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.0] - 2025-10-14

### Added

- BNDialogTitle: preset-based color variants with close icon; supports `variant="primary"` and explicit bg/text token overrides; integrates with BNDrawer via `titleProps` (bn-dialog-title, 2025-10-14)
- BNDrawer: wrapper component exposing `titleProps` to render BNDialogTitle and forward Drawer props (bn-drawer, 2025-10-14)

### Documentation

- Storybook: BNDrawer story demonstrating preset title styling, close handling, and content (bn-drawer-story, 2025-10-14)

### Build

- Changelog updated for new components and changes (changelog-update, 2025-10-14)

## [1.0.0] - 2025-09-24

### Added

- MUI-style color API for BNLogo component with support for 'default', 'white' (b278591, 2025-09-24)
- Breaking change: Removed support for 'logoFill' prop, use 'default' or 'white' instead. (5997b4d, 2025-09-24)

### Changed

- BNAppFooter component updated to use new `color="white"` prop instead of deprecated variant approach (5997b4d, 2025-09-24)
- Remove logoFill and extra color variants (2397316, 2025-09-24)
- Update fill variable (0520cca, 2025-09-24)

### Fixed

- Update fixed decimal places handling in BNNumberField to preserve trailing zeros (6b4889c, 2025-09-24)

### Documentation

- Updated BNLogo docs to reflect removal of `logoFill` prop; use `color="default" | "white"` instead (5997b4d, 2025-09-24)
- Add story and update docs (b21dcfe, 2025-09-24)
