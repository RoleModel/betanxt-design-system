# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.2] - 2026-01-30

### Fixed

- Fixed spacing and z-index for TabWithSubMenu

### Changed

- Rebuilt BNAppSwitcher to use composition instead of props
- Bumped node version to 24.13.0 and pinned all workflows to use the .node-version file

## [1.1.1] - 2025-12-11

### Fixed

- Removed color prop from `BNAppBar` and hardcoded it to primary
- Fixed `MuiAppBar` style overrides to respect base and BetaNXT themes

## [1.1.0] - 2025-11-25

### Added

- New `BNAppHeader` and related components for more flexible composition of application headers
- `ThemeToggle` component to switch between light and dark themes

## [1.0.0] - 2025-09-24

### Added

- MUI-style color API for BNLogo component with support for 'default', 'white' (b278591, 2025-09-24)
- Breaking change: Removed support for 'logoFill' prop, use 'default' or 'white' instead. (5997b4d, 2025-09-24)

### Changed

- BNAppFooter component updated to use new `color="white"` prop instead of deprecated variant approach (5997b4d, 2025-09-24)
- Remove logoFill and extra color variants (2397316, 2025-09-24)
- Update fill variable (0520cca, 2025-09-24)

### Fixed

- Update fixed decimal places handling in BNNumberField to preserve trailing zeros (6b4889c, 2025-09-24)

### Documentation

- Updated BNLogo docs to reflect removal of `logoFill` prop; use `color="default" | "white"` instead (5997b4d, 2025-09-24)
- Add story and update docs (b21dcfe, 2025-09-24)

## [0.0.58] - 2025-09-18

### Added

- Menu subheader support in BNAppBar with `menuSubheaderLabel` prop (d425b69, 2025-09-18)
- Enhanced accessibility features for BNAppBar menu navigation with proper ARIA attributes (d425b69, 2025-09-18)

### Changed

- Enhanced BNAnimatedMenuIcon with proper menu structure and navigation roles (d425b69, 2025-09-18)
- Improved drawer toggle handling with proper state management using functional updates (d425b69, 2025-09-18)
- Menu component enhanced with proper ARIA controls and navigation attributes (d425b69, 2025-09-18)
- Remove fullwidth="true" from BNTypographyPair (46f8b07, 2025-09-18)
- Remove unused export from BNTypographyPair component (e72ab87, 2025-09-12)

### Documentation

### Build
