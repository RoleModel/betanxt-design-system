# Using the Theme

## Coming Soon: npm Package

Our plan is to release the theme as an npm package in the future that will allow you to install the design system and use it in your project.

```bash
npm install @betanxt/design-system
```

## To use the theme in your app:

```jsx
// In your application's root component (e.g., App.tsx, main.tsx)
"use client";

import { ReactNode } from "react";
import { CssBaseline, ThemeProvider } from "@mui/material";

// Choose the theme you want to use:
import theme from "@betanxt/design-system/themes/baseTheme";
// Or, to use betanxtTheme:
// import theme from "@betanxt/design-system/themes/betanxtTheme";y

// Your application's root component
export default function App({ children }: { children: ReactNode }) {
  return (
    <ThemeProvider theme={theme}>
      {/* CssBaseline applies baseline styles and enables color scheme switching */}
      <CssBaseline enableColorScheme />
      {children}
    </ThemeProvider>
  );
}
```

---

## Dark Mode Support

Both `baseTheme` and `betanxtTheme` support light and dark modes automatically.

---

## Using CSS Variables (`theme.vars`)

The recommended way to access theme values is through the `theme.vars` object. This leverages native CSS variables, offering better performance and automatic responsiveness to theme changes.

```jsx
import Box from '@mui/material/Box'

// Example of using theme.vars with the sx prop on an MUI Box component
const MyStyledBox = () => (
  <Box
    sx={(theme) => ({
      backgroundColor: theme.vars.palette.primary.main,
      color: theme.vars.palette.primary.contrastText,
      padding: theme.vars.spacing[2], // Accesses 'var(--mui-spacing-2)'
      fontSize: theme.vars.typography.fontSize,
      fontFamily: theme.vars.typography.fontFamily,
      border: `1px solid ${theme.vars.palette.divider}`,
      borderRadius: theme.vars.shape.borderRadius,
      '&:hover': {
        backgroundColor: theme.vars.palette.primary.dark,
      },
    })}
  >
    This Box is styled using theme.vars via the sx prop.
  </Box>
)
```
